# Thermal Masses

The thermal masses of particles in a plasma are written as

```{math}
:label: eq-thermal-masses
m_{D}^2
= \frac{4g^2}{d_A} \int \frac{d^3p}{(2\pi)^3} \frac{1}{2p}
\left[ 
\nu_g C_A f_g(p) + \nu_q C_F \sum_f (f_q(p) + f_{\bar{q}}(p))
\right]\;,\\
m_{Q_f}^2
= g^2 C_F \int \frac{d^3p}{(2\pi)^3} \frac{1}{2p}
\left[ 
2 f_g(p) + (f_q(p) + f_{\bar{q}}(p))
\right]\;,
```

we simplify the color factors $\frac{\nu_g C_A}{d_A} = 2 N_c$ and $\frac{\nu_q C_F}{d_A} = 1$ to obtain
```{math}
:label: eq-thermal-masses-1
m_{D}^2
= 4g^2 \int \frac{d^3p}{(2\pi)^3} \frac{1}{2p}
\left[ 
2 N_c f_g(p) + \sum_f (f_q(p) + f_{\bar{q}}(p))
\right]\;,
```
