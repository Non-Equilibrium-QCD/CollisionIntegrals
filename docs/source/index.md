# CollIntegral

QCD Collision Integral Computation Library.

## Overview

This library computes collision integrals for QCD processes using Monte Carlo integration (GSL VEGAS).

## Features

- Multiple integration backends (GSL VEGAS)
- OpenMP parallelization
- Generic template-based collision integral framework
